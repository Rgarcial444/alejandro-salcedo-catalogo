# Alejandro Salcedo - Catálogo de Autos
Sitio de asesoría automotriz.
## Características
- Inventario de vehículos desde Supabase
- Filtros por marca, precio, condición
- Detalle de auto con galería
- Formulario de contacto
- WhatsApp flotante
- Admin para gestionar vehículos
- Cambio de foto de perfil
